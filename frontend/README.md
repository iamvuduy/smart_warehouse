# Frontend - AI Smart Warehouse Optimization (SLAP)

This is a Vite + React frontend using TailwindCSS.

Install dependencies:

```
cd frontend
npm install
```

Run dev server (either):

```
# preferred during development
npm run dev

# or you can use npm start (maps to the same command)
npm start
```

Open the app in your browser (default Vite port 5173). Ensure the backend FastAPI server is running on http://localhost:8000
